# SophosMap 🏛️

Aplicación desktop interactiva de línea temporal de filósofos.

## Estructura
- `backend/`: API en Node.js + Express + PostgreSQL
- `frontend/`: Aplicación en React + JavaScript

## Instalación
1. **Backend**:
   ```bash
   cd smbackend
   npm install
   cp .env.example .env
   npm run dev

2. **Frontend**:
   ```bash
   cd smfrontend
   npm install
   npm run dev
